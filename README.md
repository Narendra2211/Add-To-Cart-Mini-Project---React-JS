# Add-To-Cart-Mini-Project---React-JS
The Add to Cart mini project is a simple React-based application that allows users to add items to a shopping cart, view the cart list, and remove items using their index. It demonstrates React Hooks, state management, and basic UI rendering.

This project is ideal for beginners learning useState, useRef, event handling, and list rendering in React.
